# Saqlain Abbas - Portfolio

A brutalist-style personal portfolio with a terminal-inspired UI and monospace typography.

## Design
- High contrast (black & white)
- Monospace fonts only
- Sharp layouts, minimal animations
- Terminal/CLI inspired interface
- Content-first approach

Inspired by Polar.sh and Linear.

## Features

### Hero
- Typewriter text effect
- Blinking cursor
- CLI-style buttons

### About
- Terminal info blocks
- Numbered cards
- Simple grid layout

### Projects
- 2-column grid
- Tech stack tags
- GitHub links
- Hover effects

### Contact
- Terminal-style form
- Formspree integration
- Keyboard shortcuts

### Navigation
- Fixed navbar
- Numbered sections
- Smooth scroll
- Mobile menu

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Formspree
- GitHub Pages

## Setup

```bash
npm install --legacy-peer-deps
npm run dev
npm run build
npm run preview
npm run deploy
```

## Structure

```
portfolio/
├── src/components/
│   ├── HeroSection.tsx
│   ├── Navbar.tsx
│   ├── AboutSection.tsx
│   ├── ProjectsSection.tsx
│   └── ContactSection.tsx
├── App.tsx
├── index.css
└── main.tsx
```

## Customization
- Update Formspree ID in `ContactSection.tsx`
- Add project images in project cards
- Edit navigation items in `Navbar.tsx`
